# 📊 Sales & Finance Analytics Project – Excel


# 🔎 Project Overview

This project showcases an end-to-end Sales and Finance Analytics solution built using Microsoft Excel’s advanced analytics capabilities.

The objective was to transform raw transactional sales and financial data into structured, insight-driven business reports that enable strategic decision-making.

Using Power Query for ETL, Power Pivot for data modeling, and Pivot Tables for reporting, I designed a scalable analytical solution based on a Star Schema model.

This project demonstrates practical implementation of data cleaning, modeling, performance tracking, and financial analysis using real-world business scenarios.

# 🎯 Business Problem

Organizations often struggle with:

Identifying top revenue-generating customers

Tracking market performance against sales targets

Monitoring profitability trends over time

Converting raw data into actionable insights

This project addresses these challenges by building interactive analytical reports that answer:

Which customers drive the highest revenue?

Which markets are exceeding or missing targets?

How does profitability trend across fiscal periods?

What patterns impact financial performance?

This project follows a structured Data Analytics Workflow:

Business Insight Generation

This ensures consistent, validated, and analysis-ready data.

# 🗂 Data Model (Star Schema)

A structured Star Schema model was created to optimize analytical performance.

📌 Fact Table

Sales Transactions

📌 Dimension Tables

Customers

Products

Markets

Date (Fiscal Calendar)

This structure enables multi-dimensional analysis across time, geography, and product categories.
